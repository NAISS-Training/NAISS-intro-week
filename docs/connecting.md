# Connecting and File transfers  

**Time**: 13:00 - 16:00 

This course will teach you how to login to and transfer files between NAISS HPC clusters.

!!! note

    Click the Home (at the top) to see the date.

## Prerequisites

- Prerequisite 1/5: (Windows users only) Have MobaXterm installed
- Prerequisite 2/5: (Mac and Linux users only) Have rsync installed
- Prerequisite 3/5: Have FileZilla installed. (if you cannot install, see the FAQ 'What if I cannot install FileZilla?')
- Prerequisite 4/5: Be able to find the course's Zoom room's URL on your HPC cluster
- Prerequisite 5/5: have a good Zoom setup

More here: <a href="https://uppmax.github.io/naiss_file_transfer_course/prereqs/" target="_blank">prerequisites</a>. 

## Topics 

- Login with terminal
- File transfer using scp
- File transfer using FileZilla
- File transfer using rsync
- Cluster to cluster
- File transfer using sftp
