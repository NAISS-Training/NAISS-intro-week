# Linux Command Line 101

**Time**: 9:00 - 12:00

This NAISS course is an introduction to Linux, which is what is used on most Swedish HPC centers and indeed on the vast majority of HPC clusters worldwide. 
It aims to give a brief, but comprehensive introduction to the Linux operating system.

It is meant for **beginners** to the Linux operating system.

!!! note 

    Click the Home (at the top) to see the date. 

## Prerequisites

- You need access to a Linux system.
    - You can use your own computer or any other computer (including computing access at computing centers) you have access to and that runs Linux. 

## Topics 

- What is Linux and why should I use it?
- You will learn about
    - the command line interface (CLI)
    - navigating and modifying the file tree
        - ls
        - mkdir
        - cd
        - mv
    - editors on the Linux system
    - pipes, sorting, filters
        - redirection
        - pipes
        - etc.
    - patterns
    - scripting
    - some handy hints and tricks

This course will consist of lectures and type-alongs, as well as a few exercises where you get to try out what you have just learned.
