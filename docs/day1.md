## Linux Command Line 101 

**Time**: 9:00 - 12:00

This NAISS course is an introduction to Linux, which is what is used on most Swedish HPC centers and indeed on the vast majority of HPC clusters worldwide. 
It aims to give a brief, but comprehensive introduction to the Linux operating system.

It is meant for **beginners** to the Linux operating system.

### Prerequisites

- You need access to a Linux system.
    - You can use your own computer or any other computer (including computing access at computing centers) you have access to and that runs Linux. 

### Topics 

- What is Linux and why should I use it?
- You will learn about
    - the command line interface (CLI)
    - navigating and modifying the file tree
        - ls
        - mkdir
        - cd
        - mv
    - editors on the Linux system
    - pipes, sorting, filters
        - redirection
        - pipes
        - etc.
    - patterns
    - scripting
    - some handy hints and tricks

This course will consist of lectures and type-alongs, as well as a few exercises where you get to try out what you have just learned.

## Connecting and File transfers  

**Time**: 13:00 - 16:00 

This course will teach you how to login to and transfer files between NAISS HPC clusters.

### Prerequisites

- Prerequisite 1/5: (Windows users only) Have MobaXterm installed
- Prerequisite 2/5: (Mac and Linux users only) Have rsync installed
- Prerequisite 3/5: Have FileZilla installed. (if you cannot install, see the FAQ 'What if I cannot install FileZilla?')
- Prerequisite 4/5: Be able to find the course's Zoom room's URL on your HPC cluster
- Prerequisite 5/5: have a good Zoom setup

More here: <a href="https://uppmax.github.io/naiss_file_transfer_course/prereqs/" target="_blank">prerequisites</a>. 

### Topics 

- Login with terminal
- File transfer using scp
- File transfer using FileZilla
- File transfer using rsync
- Cluster to cluster
- File transfer using sftp
